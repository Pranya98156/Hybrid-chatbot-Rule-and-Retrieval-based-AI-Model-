# 🤖 Hybrid Chatbot for Academic & Administrative Use
 
  Project Overview

This project implements a **Hybrid Chatbot** that combines:

* Rule-based approach (for predefined queries)
* Retrieval-based approach (using TF-IDF and cosine similarity)

It is designed to assist users with:

* Admissions
* Fees
* Courses
* Exams
* General university queries



## 🧠 Methodology

1. User Input
2. Text Preprocessing
3. Intent Detection
4. Routing:

   * Rule-based system
   * Retrieval-based system
5. Response Generation

---

## ⚙️ Technologies Used

* Python
* Scikit-learn
* Natural Language Processing (NLP)
* Google Colab

---

## 📊 Evaluation Metrics

Accuracy = Correct Responses / Total Queries
